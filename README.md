# Social App 🚀

A full-stack social media application built using modern web technologies.

## 🔧 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## ✨ Features
- User Authentication (Login/Register)
- Create & View Posts
- User Feed
- Secure Backend APIs

## 📁 Project Structure
social-app/
│── backend/
│── frontend/

## 🚀 How to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
